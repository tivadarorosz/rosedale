## Project Overview
Rosedale Massage is a premium massage studio dedicated to providing exceptional services in Guildford. This repository supports the studio's operations, including:
- **Email Templates**: For client and therapist communications.
- **Marketing Campaigns**: Flyers, newsletters, and other promotional materials.
- **Appointment Booking System**: Integration and customization.
- **Website Assets**: Code and content for the official studio website.

---

## Features
- **Email Templates**: HTML templates for customer notifications, therapist updates, and marketing.
- **Custom Booking System**: Features such as waiting lists and automatic notifications.
- **Marketing Materials**: Flyers, social media campaigns, and other creative assets.
- **Scalable Folder Structure**: Organized files for ease of collaboration and future scaling.

## Getting Started
To use or contribute to this project:

1. Clone the repository:
   ```bash
   git clone https://github.com/tivadarorosz/rosedale.git
   ```
2. Navigate to the project directory:
   ```bash
   cd rosedale
   ```
3. Install dependencies for the booking system (if applicable):
   ```bash
   npm install
   ```
4. Run local development:
   ```bash
   npm start
   ```
